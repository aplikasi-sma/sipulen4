      Pencadangan Data

      Ekspor, impor, dan pemulihan data sistem secara menyeluruh.

      Ekspor Data

      Unduh seluruh data (Siswa, Guru, Riwayat, Pelanggaran) ke dalam satu file .json aman.

      Impor Pemulihan

      Pulihkan sistem dengan mengunggah file .json hasil ekspor sebelumnya. Data saat ini akan tertimpa.

      Kosongkan Sistem

      Hapus seluruh data siswa dan riwayat secara permanen. Pengaturan guru & jenis pelanggaran akan kembali ke bawaan.
